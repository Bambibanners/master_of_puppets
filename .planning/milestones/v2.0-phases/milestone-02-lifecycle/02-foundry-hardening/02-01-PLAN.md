  <name>Task 2: Offload File Operations to Threads</name>
  <files>puppeteer/agent_service/services/foundry_service.py</files>
  <behavior>
    - Use `asyncio.to_thread` for `os.makedirs`, `shutil.copytree`, `shutil.copy2`, and `shutil.rmtree`.
  </behavior>
  <action>
    Modify `FoundryService.build_template` to use `asyncio.to_thread`.

    ```python
    # Example changes:
    await asyncio.to_thread(os.makedirs, build_dir, exist_ok=True)
    
    if os.path.isdir(env_src):
        await asyncio.to_thread(shutil.copytree, env_src, env_dst, dirs_exist_ok=True)
    
    if os.path.isfile(req_src):
        await asyncio.to_thread(shutil.copy2, req_src, req_dst)
        
    # In finally:
    if os.path.exists(build_dir):
        await asyncio.to_thread(shutil.rmtree, build_dir)
    ```
  </action>
  <verify>

  </verify>
  <done>Blocking file operations are offloaded from the main event loop</done>
